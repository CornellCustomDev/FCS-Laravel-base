# Laravel Boost

This repo is set up to support running [Laravel Boost](https://laravel.com/ai/boost).

## Installation

To get everything set up initially, run:

```bash
lando artisan boost:install
```

The [`boost.json`](../boost.json) and [`config/boost.php`](../config/boost.php) define the install defaults, 
which you can accept as-is or modify for your project. The config file mostly helps with using Boost with Lando.
